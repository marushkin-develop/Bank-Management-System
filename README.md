# Система управления банком (Bank Management System)

Одностраничное веб-приложение, имитирующее работу банковской системы с ролями клиента и администратора. Проект основан на описании Java-приложения, но полностью реализован на HTML, CSS и JavaScript для лёгкого развёртывания на статических хостингах вроде Netlify.

## 🔧 Основные возможности

### 👤 Для клиентов
- Регистрация нового счёта (требует подтверждения администратором)
- Проверка текущего баланса
- Внесение наличных
- Снятие наличных с контролем достаточности средств
- Перевод средств другому клиенту по номеру счёта
- Просмотр истории собственных транзакций

### 👑 Для администраторов
- Сводка по системе: число активных счетов, общая сумма депозитов, количество ожидающих заявок
- Утверждение или отклонение новых регистраций
- Просмотр всех счетов с фильтрацией по статусу (активен, ожидает, заблокирован)
- Блокировка и разблокировка пользователей
- Общий журнал транзакций всех клиентов

## 🛠️ Технические детали

- **Языки:** HTML5, CSS3, JavaScript (ES6, без фреймворков)
- **Хранение данных:** `localStorage` браузера (все счета, транзакции и пользователи сохраняются локально)
- **Архитектура:** полностью клиентское приложение, не требует серверной части
- **Безопасность:** проверка корректности операций, предотвращение перевода самому себе, блокировка неактивных счетов
- **Адаптивный дизайн:** интерфейс корректно отображается на мобильных устройствах
- **Совместимость:** работает во всех современных браузерах (Chrome, Firefox, Safari, Edge)

### 🧪 Демонстрация работы
1. Откройте развёрнутое приложение [https://fluffy-rugelach-e755f7.netlify.app/](https://fluffy-rugelach-e755f7.netlify.app/)
2. Войдите как администратор, используя логин `admin` и пароль `admin123`.
3. В панели администратора ознакомьтесь со сводкой по системе.
4. Зарегистрируйте новый клиентский счёт через кнопку «Зарегистрироваться» на экране входа.
5. Вернитесь в профиль администратора, перейдите во вкладку «На одобрение» и подтвердите заявку.
6. Войдите под новым клиентом и выполните операции:
   - Пополните счёт.
   - Снимите наличные.
   - Переведите деньги другому счёту (например, другому зарегистрированному клиенту).
7. Проверьте историю транзакций — она обновляется мгновенно.
8. Администратор может просмотреть общий журнал транзакций и управлять статусами счетов.
